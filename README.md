# Projet-SQL-
# 🗄️ Mini Projet SQL – Analyse et gestion de base de données

##  Présentation du projet
Ce projet a été réalisé dans le cadre du module **Systèmes de Gestion de Bases de Données (SQL)**.  
L’objectif est de mettre en pratique les concepts fondamentaux du SQL à travers :

- l’analyse d’une base de données existante,
- la création d’une base de données complète,
- l’insertion et la manipulation des données,
- l’écriture de requêtes SQL simples et avancées.

Le projet est divisé en deux parties principales :  
- Analyse d’une base existante (films, clients, paiements, etc.)  
- Création et gestion d’une nouvelle base de données 

---

## Objectifs pédagogiques

Ce projet permet de développer les compétences suivantes :

- Écriture de requêtes SQL (SELECT, WHERE, ORDER BY)
- Utilisation des fonctions d’agrégation (SUM, AVG, COUNT)
- Utilisation des jointures (JOIN)
- Création de vues (VIEW)
- Création de bases de données et de tables
- Insertion et suppression de données
- Création d’index pour optimiser les performances
- Analyse de données avec SQL
